from [this steam deck tutorial](https://www.reddit.com/r/SteamDeck/comments/16kwzbi/comment/kg6cer4/)

 Extract https://www.cwdohnal.com/utglr/dxglr21.zip into 
 ```
 ~/.local/share/Steam/steamapps/common/Deus Ex/System/
``` 
 Pick the OpenGL renderer on the first run. Set the control layout to "Deus Ex Steam Deck Perfection". 
 In game, set the resolution to 768x480 for larger text.
